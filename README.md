# 장기렌트 DB 수집 랜딩 페이지

이 프로젝트는 신차 장기렌트/리스 최적가 비교 및 DB 수집을 위한 랜딩 페이지입니다.

## 주요 기능
- **수수료 0% 직판 견적**: 영업사원 수수료를 100% 제거한 다이렉트 최적가 견적.
- **간편한 문의 폼**: 사용자 연락처와 차종 정보를 빠르게 입력하고 문의할 수 있는 Lead Form.
- **반응형 웹 디자인**: 모바일과 데스크톱 모두에서 뛰어난 사용자 경험을 제공.

## 기술 스택
- **프레임워크**: Next.js 16 (App Router)
- **스타일링**: Tailwind CSS
- **배포**: Netlify (또는 Vercel)

## 시작하기

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 접속 후 확인.
